📖 Révisions express – CM2 vers 6ème

🇫🇷 Français – Essentiel

  • Présent : je mange, tu manges, il/elle mange, nous mangeons, vous mangez, ils/elles mangent
  • Futur : je mangerai, tu mangeras, il/elle mangera, nous mangerons, vous mangerez, ils/elles mangeront
  • Imparfait : je mangeais, tu mangeais, il/elle mangeait, nous mangions, vous mangiez, ils/elles mangeaient
  • Passé composé : j'ai mangé, tu as mangé, il/elle a mangé, nous avons mangé, vous avez mangé, ils/elles ont mangé
  • Accord du participe passé : avec être → accord avec le sujet ; avec avoir → accord avec COD placé avant
  • Nature des mots : nom, verbe, adjectif, déterminant, pronom, adverbe, préposition, conjonction
  • Fonctions : sujet, verbe, complément d'objet direct (COD), complément d'objet indirect (COI), complément circonstanciel

🔢 Maths – Essentiel

  • Addition : poser les chiffres alignés par la droite, retenues
  • Soustraction : poser aligné par la droite, emprunts
  • Multiplication : poser, multiplier par chaque chiffre, additionner les résultats partiels
  • Division : dividende ÷ diviseur = quotient, reste possible
  • Fractions : ½ = 0,5 ; ¼ = 0,25 ; ¾ = 0,75 ; additionner avec même dénominateur
  • Décimaux : 3,5 + 2,8 → additionner les dixièmes (5+8=13 → 1 unité + 3 dixièmes)
  • Conversions : 1 km = 1000 m ; 1 m = 100 cm ; 1 kg = 1000 g ; 1 L = 100 cL = 1000 mL
  • Périmètre : rectangle = 2×(L+l) ; carré = 4×côté
  • Aire : rectangle = L×l ; carré = côté×côté

⚠️ Pièges fréquents

  • Accords : "Les enfants jouent" (pas "joue") – sujet pluriel = verbe pluriel
  • Homophones : "à" (préposition) vs "a" (avoir) ; "et" (conjonction) vs "est" (être) ; "on" (pronom) vs "ont" (avoir)
  • Fractions : ne pas oublier de mettre au même dénominateur avant d'additionner
  • Décimaux : 3,5 + 2,8 = 6,3 (pas 7,3 !)
  • Conjugaison : "nous mangeons" (garder le 'e') ; "ils mangent" (pas "mangent" avec un 't' supplémentaire)
